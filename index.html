Survival

How the world actually works

Vanilla-first survival with Fabric mods that earn their keep. No pay-to-win, no land claims you buy, no grief protection you configure—just a whitelist, common sense, and ops who show up when things go sideways.

Difficulty & gameplay

  • Hardcore survival on Easy — mobs hit, resources matter, but you are not farming iron for three hours to afford a sword.
  • Survival mode only — no creative, no spectator, no /gamemode for players. Ops use creative for builds and rollbacks only.
  • Whitelist enforced — white-list=true + enforce-whitelist=true in server.properties. If you are not on the list, you get kicked.
  • Force gamemode — force-gamemode=true so you cannot accidentally tab into creative.

Base building & griefing

  • No land claim plugins — this is a whitelist server. We trust each other. If someone griefs, they are off the list permanently.
  • Build anywhere — the world is yours. Just respect other players' builds and do not tear down what is not yours.
  • Backups are real — world backups run on a schedule. If something goes wrong, we restore from the last clean save.
  • Common sense rule — if you have to ask whether it is grief, it probably is. Just ask first.

Server uptime & restarts

  • 20 TPS target — if the server drops below 18 for more than a few minutes, ops investigate.
  • Scheduled restarts — the server restarts periodically to clear memory leaks and keep chunk loading smooth.
  • Ops announce restarts — you get a warning in chat before the server goes down. Save your stuff.
  • Uptime is a goal, not a religion — we are not a 24/7 hosting company. The server runs when the host machine is on.

Community & etiquette

  • Small group, high trust — this is not a public SMP. You are here because someone vouched for you.
  • Chat is logged — RCON captures all chat. Ops can review if there is a dispute.
  • Respect the world — no massive strip mines near spawn, no lava casts on other people's hills, no redstone clocks that tank TPS.
  • Discord for coordination — server chat is for in-game comms. Discord is for planning, screenshots, and "is the server up?"
